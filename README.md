# 🌦️ Weather Forecast Web App

A modern and interactive Weather Forecast Web Application built using **HTML, CSS, and JavaScript**.  
It fetches real-time weather data using the **OpenWeatherMap API** and displays current weather, 5-day forecast, animated backgrounds, and more.

---

## 🚀 Features

- 🌍 **Search Weather by City Name**
- 📍 **Get Weather Using Current Location**
- 🌡️ **Celsius ↔ Fahrenheit Unit Toggle**
- 🕒 **Live Sunrise & Sunset Time**
- 🌧️ **Dynamic Weather Backgrounds**
- ❄️ **Rain & Snow Canvas Animations**
- 📅 **5-Day Forecast Display**
- 🔄 **Auto Refresh Smooth UI**
- 🎞️ **Video Background Support**

---

## 🛠️ Technologies Used

- **HTML5**  
- **CSS3**  
- **JavaScript (ES6+)**
- **OpenWeatherMap API**
- **Canvas API (for animations)**
